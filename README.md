# Spy VK Mini-App

Игра "Шпион" в ВК

https://vk.com/app8197123

## Фичи

- Список локаций
- Ввод количества игроков
- Выбор шпионов в соответствии с количеством игроков
- Незаметная раздача карт
- Таймер раунда
- Сигналы вспышкой (если имеется) при окончании раунда
- Ввод имен игроков
- Кастомизация локаций
